# Android Basics with Jetpack Compose - Course Projects

This repository contains my solutions and projects built while progressing through the [Android Basics with Jetpack Compose](https://developer.android.com/courses/android-basics-compose/course) course by Google. 

Each project is organized into its own feature branch corresponding to the course units and pathways.

## 📌 Repository Structure & Branches

Instead of keeping everything on a single branch, each app is developed on a separate branch to keep the codebase clean and modular. You can switch between branches to view the specific project code.

### 🔹 Unit 1: Your First Android App
* **`Unit-1/firstapp`** - Introduction to Android Studio, building a basic user interface, and running a first "Hello World" style application.
* **`Unit-1/Business-card`** - A personal practice project creating a digital business card layout using text, icons, and arrangements in Jetpack Compose.

### 🔹 Unit 2: Building App UI
* **`Uniy-2/Dice-roller`** - A dynamic app that simulates rolling a dice. Introduces the fundamentals of State (`mutableStateOf`, `remember`) and handling user interaction via button clicks.
* **`Uniy-2/Lemonade`** - A state-driven interactive app that walks the user through the process of making lemonade, reinforcing conditional logic tied to UI state.
* **`Uniy-2/TipTime`** - A functional tip calculator that introduces text input fields, keyboard configurations, state hoisting, and writing automated local unit tests.
* **`Uniy-2/ArtSpaceApp`** - A creative assignment showcasing an artwork gallery with interactive "Next" and "Previous" navigation controls to practice state management.

### 🔹 Unit 3: Displaying Lists and Styling Apps
* **`Unit-3/AffirmationApp`** - Displays a scrollable list of text and images using `LazyColumn`, focusing on data models and performance-optimized lists.
* **`Unit-3/GridApp`** - Implements a scrollable grid layout using `LazyVerticalGrid` to display topics or categorized information neatly.
* **`Unit-3/WoofApp`** - A beautifully styled application displaying a list of dogs. Explores Material Design 3 guidelines, custom themes, shapes, typography, and simple explicit animations (`animateContentSize`).
* **`Unit-3/SuperHeroesApp`** - A practice project implementing a clean list layout of superheroes, reinforcing Material 3 customization, custom typography, and card designs.

### 🔹 Unit 4: Architecture and App Navigation
* **`Unit-4/UnscrambleGame`** - Built during Pathway 1 (State Management in Compose). Focuses on modern Android architecture by separating UI from business logic using a **ViewModel** and **StateFlow** within a word unscrambling game.

## 🚀 Technologies Used
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Material 3)
* **Development Environment:** Android Studio
* **Architecture Concepts:** State Management, Composable Functions, Modifiers, State Hoisting, and Material Design themes.
